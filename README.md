# 💡 IdeaFlow – Share & Exchange Ideas

## 🚀 Overview

**IdeaFlow** is a MERN stack web application designed to help users share, discover, and exchange ideas in a collaborative environment. It provides a simple and interactive platform for expressing thoughts, engaging in discussions, and connecting with like-minded people.

---

## ✨ Features

* 📝 Post and share ideas
* ❤️ Like and react to posts
* 💬 Comment and discuss ideas
* 🔐 Secure user authentication (Login/Register)
* 👤 User profiles
* 🌐 Responsive and user-friendly interface

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* HTML5, CSS3, JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/ideaflow.git
cd ideaflow
```

### 2️⃣ Install dependencies

```
cd server
npm install

cd ../client
npm install
```

### 3️⃣ Run the application

```
# Start backend
cd server
npm start

# Start frontend
cd client
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder and add:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
